# HTN001 Case Report Form (CRF)

## Purpose

This folder contains the Case Report Form (CRF) developed for the HTN001 mock clinical study.

## Study Information

- **Study ID:** HTN001
- **Study Design:** Phase II, Randomized, Double-Blind, Placebo-Controlled
- **Indication:** Essential Hypertension
- **Sample Size:** 120 Subjects
- **Treatment Arms:** Drug A 50 mg and Placebo

## CRF Sections

- Cover Page
- Informed Consent
- Demographics
- Inclusion/Exclusion Criteria
- Medical History
- Concomitant Medications
- Treatment Administration
- Vital Signs
- Laboratory Assessments
- Adverse Events
- End of Study
