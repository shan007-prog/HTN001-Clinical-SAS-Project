# SDTM Datasets

## Overview

This folder contains the finalized SDTM datasets generated for the HTN001 mock clinical study.

These datasets represent standardized clinical data derived from the raw database and serve as the source for ADaM dataset creation.

## Available Datasets

* DM – Demographics
* EX – Exposure
* DS – Disposition

Additional domains will be added as programming progresses.

## Purpose

The SDTM datasets provide:

* Standardized subject-level clinical data
* Consistent variable naming
* CDISC-compliant structure
* Source data for analysis dataset derivation
