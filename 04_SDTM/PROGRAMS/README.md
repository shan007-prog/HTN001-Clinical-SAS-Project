# SDTM Programs

This folder contains SAS programs used to create SDTM domains from the raw clinical database.

Each program includes:

* Program header
* Input datasets
* SDTM mapping logic
* Variable derivations
* Controlled terminology implementation
* Output SDTM dataset

Current domains include:

* DM – Demographics
* EX – Exposure
* DS – Disposition
* AE – Adverse Events
* CM – Concomitant Medications
* VS – Vital Signs
* LB – Laboratory
* MH – Medical History

Programming follows CDISC SDTM Implementation Guide standards.
