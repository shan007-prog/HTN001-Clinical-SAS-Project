# SDTM Quality Control

## Overview

This folder contains validation programs and supporting documentation used to verify the accuracy of the HTN001 SDTM datasets.

## Validation Performed

* Record count verification
* Duplicate subject checks
* Missing value checks
* Variable order verification
* ISO 8601 date validation
* PROC COMPARE validation
* Programming review

## Objective

To ensure that each SDTM domain accurately reflects the source data and complies with CDISC SDTM standards before being used for ADaM programming.
