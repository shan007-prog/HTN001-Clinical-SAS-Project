# ADaM Datasets

## Overview

This folder contains the finalized analysis datasets for the HTN001 study.

The datasets are derived from SDTM and structured according to CDISC ADaM principles to support statistical analyses.

## Current Dataset

* ADSL – Subject-Level Analysis Dataset

Additional ADaM datasets will be added as the project progresses.

## Purpose

The ADaM datasets provide analysis-ready data for:

* Demographic summaries
* Subject disposition
* Safety analyses
* Efficacy analyses
* Statistical reporting
