# HTN001 Clinical SAS Portfolio Project

## Project Overview

HTN001 is an end-to-end mock Clinical SAS portfolio project developed to demonstrate the clinical trial programming workflow:

**Protocol → CRF → Mock Raw Data → SDTM → ADaM → Tables, Listings and Figures**

The study is a randomized, double-blind, placebo-controlled Phase II clinical trial evaluating Drug A 50 mg versus placebo in subjects with essential hypertension.

## Study Design

* **Study ID:** HTN001
* **Phase:** Phase II
* **Design:** Randomized, double-blind, placebo-controlled
* **Sample Size:** 120 subjects
* **Treatment Groups:** Drug A 50 mg and Placebo
* **Treatment Duration:** 12 weeks
* **Primary Endpoint:** Change from baseline in systolic blood pressure at Week 12

## Repository Structure

* `01_Protocol` – Mock clinical study protocol
* `02_CRF` – Case Report Form
* `03_Raw_Data` – Synthetic raw clinical datasets
* `04_SDTM` – SDTM specifications, SAS programs, QC and outputs
* `05_ADaM` – ADaM specifications, SAS programs, QC and outputs
* `06_TLF` – TLF shells, programs, QC and report outputs
* `08_Study_Documents` – Statistical Analysis Plan and supporting documents

## Programming Skills Demonstrated

* SAS DATA step programming
* `PROC SORT`, `PROC SQL`, `PROC FREQ`, `PROC TRANSPOSE` and `PROC REPORT`
* Dataset merging using `BY` groups and `IN=` variables
* ISO 8601 date conversion
* SDTM study-day derivations
* ADaM treatment and population flag derivations
* Duplicate, missing-value and consistency checks
* Independent QC and `PROC COMPARE`
* RTF table generation
